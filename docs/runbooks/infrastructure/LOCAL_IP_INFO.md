## 注意事項

⚠️ **IP 地址可能變化**
- 如果重新連接 WiFi 或網絡，IP 可能會改變
- 建議定期檢查 IP 地址
- 可以使用 `ifconfig` 或 `ipconfig` 命令查看

⚠️ **防火牆設置**
- 確保防火牆允許端口 3000 的連接
- macOS: 系統設置 → 網絡 → 防火牆

⚠️ **網絡安全**
- 僅在開發環境使用
- 生產環境應使用 HTTPS 和域名
